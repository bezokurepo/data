DELTACORPUS - 107 languages automatically tagged by a DELexicalized TAgger

License: CC BY-SA 4.0

Web page: http://ufal.mff.cuni.cz/deltacorpus

For details, see the paper:
Zhiwei Yu, David Mareček, Zdeněk Žabokrtský, Daniel Zeman: If You Even Don’t Have a Bit of Bible: Learning Delexicalized POS Taggers. In Proceedings of LREC 2016, Portorož, Slovenia, May 2016

Changelog:

version 1.0, 2016-03-22
* All languages tagged by an SVM classifier trained on the c7 mix (bg, ca, de, el, hi, hu, tr).
* Google Universal POS tag set (12 tags).
* Trained on HamleDT 2.0.

version 1.1, 2016-06-20
* Slavic, Germanic and Romance languages tagged by classifier trained on the respective language group only.
* Other languages tagged by classifier trained on all available languages.
* Baltic languages treated as Slavic, Hungarian as Germanic, Romanian as other.
* Universal Dependencies POS tag set (17 tags).
* Trained on Universal Dependencies 1.2.
